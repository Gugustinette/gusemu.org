# ROM dumping

ROM dumping is the process of extracting the data from a physical game cartridge or disc and creating a digital copy of it, which can then be used with emulators.

This process is essential for preserving games and making them accessible to a wider audience.

Here are some popular hardware for dumping ROMs:

- [Retrode](https://www.retrode.com/en/welcome/): Sega Genesis/Mega Drive and SNES
- [GBxCart RW](https://www.gbxcart.com/): Game Boy, Game Boy Color and Game Boy Advance
- [Open Source Cartridge Reader](https://github.com/sanni/cartreader): NES, SNES, Nintendo 64, Game Boy Color, Game Boy Advance, Sega Genesis/Mega Drive, and Sega Master System
