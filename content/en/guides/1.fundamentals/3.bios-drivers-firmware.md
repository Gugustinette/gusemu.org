# BIOS, drivers and firmware

Some emulators require you to obtain and use the original BIOS, drivers, or firmware from the console to function properly.

These files contain proprietary code that is protected by copyright law, and using them without permission may be illegal.

## What file do I need ?

Here are some details about the files you might need for popular consoles:

### PlayStation 1

To run the ISOs of PlayStation 1 games, you will need the BIOS file corresponding to the region of the game you want to play:

- **scph5500.bin**: Japanase BIOS
- **scph5501.bin**: US BIOS
- **scph5502.bin**: European BIOS

See more details [here](https://en.wikipedia.org/wiki/PlayStation_models#Comparison_of_models) and [here](https://consolemods.org/wiki/PS1:PS1_Model_Differences).

### PlayStation 2

To run the ISOs of PlayStation 2 games, you will need the BIOS file corresponding to the region of the game you want to play:

- **ps2-0250j-20100415.bin**: Japanese BIOS
- **ps2-0230a-20080220.bin**: US BIOS
- **ps2-0250e-20100415.bin**: European BIOS

See more details [here](https://consolemods.org/wiki/PS2:Model_Differences).

### PlayStation 3

- **PS3UPDAT.PUP (PS3 Firmware 4.92)**

See more details [here](https://en.wikipedia.org/wiki/PlayStation_3_models#Model_comparison).

### Xbox

- **mcpx_1.0.bin**: MCPX Boot ROM Image
- **Complex_4627.bin**: Flash ROM Image (BIOS)
- **xbox_hdd.qcow2**: Hard Disk Image

See more details [here](https://xemu.app/docs/required-files/).
