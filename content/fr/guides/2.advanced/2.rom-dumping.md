# ROM dumping

Le ROM dumping désigne le processus d'extraction des données d'une cartouche ou d'un disque de jeu physique pour créer une copie numérique de celui-ci, qui peut ensuite être utilisée avec des émulateurs.

Ce processus est essentiel pour préserver les jeux et les rendre accessibles à un public plus large.

Voici quelques outils populaires pour le dumping de ROMs :

- [Retrode](https://www.retrode.com/en/welcome/): Sega Genesis/Mega Drive et SNES
- [GBxCart RW](https://www.gbxcart.com/): Game Boy, Game Boy Color et Game Boy Advance
- [Open Source Cartridge Reader](https://github.com/sanni/cartreader): NES, SNES, Nintendo 64, Game Boy Color, Game Boy Advance, Sega Genesis/Mega Drive, et Sega Master System
