# ROM hacking et traductions

Le [ROM hacking](/fr/guides/ressources/glossary#rom-hacking) est le processus de modification d'une ROM de jeu vidéo pour changer son contenu, comme les visuels, la musique, le texte ou les mécaniques de jeu.

Cela peut être fait pour diverses raisons, telles que la correction de bugs, l'ajout de nouvelles fonctionnalités, ou la création de [traductions](/fr/guides/ressources/glossary#translation-patch) de jeux qui n'ont jamais été publiés dans certaines régions.

## Les fichiers de patch

En règle générale, le ROM hacking se fait en appliquant des [patchs](/fr/guides/ressources/glossary#patch) à des ROMs d'origines.

Les patchs sont des fichiers qui contiennent les différences entre la ROM d'origine et la version modifiée.

Ils sont généralement dans les formats suivants :

- `.ips` (International Patching System) : l'un des formats de patch les plus anciens et les plus largement utilisés.
- `.ups` (Universal Patching System) : un format de patch delta qui inclut fréquemment des vérifications de checksum pour garantir l'intégrité de la ROM d'origine avant d'appliquer le patch.
- `.bps` (Binary Patch Support) : un format plus récent conçu pour améliorer l'IPS, qui inclut souvent des vérifications de checksum.
- `.xdelta` / `.vcdiff` : un format de diff binaire puissant, souvent utilisé pour les changements importants.
- `.ppf` (PlayStation Patch Format) : souvent utilisé pour le patching d'images de disque.

Plus d'informations sur les formats [ici](https://rompatcher.app/patch-formats/) et [ici](https://igir.io/roms/patching/#patch-types)

## Outils

Vous avez besoin d'un outil de patching pour appliquer des patchs à des ROMs. Ces outils prennent la ROM d'origine et le fichier de patch en entrée, et produisent une nouvelle ROM avec les modifications appliquées.

Voici quelques outils populaires pour appliquer des patchs à des ROMs :

- [Rom Patcher JS](https://www.romhacking.net/patch/)
- [Xdelta patcher](https://kotcrab.github.io/xdelta-wasm/)

## Où trouver des patchs

Voici quelques sources populaires pour les ROM hacks :

- https://www.romhacking.net/hacks/
- https://vimm.net/romhacking/hacks
- https://gamebanana.com/games

Et pour les traductions :

- https://www.romhacking.net/translations/
- https://vimm.net/romhacking/translations

Aussi, si vous cherchez un jeu spécifique (ou une traduction spécifique), essayez de rechercher "nom du jeu + rom hack" ou "nom du jeu + translation" sur votre moteur de recherche préféré, vous pourriez trouver des pépites cachées !
