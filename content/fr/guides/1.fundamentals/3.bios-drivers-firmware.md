# BIOS, drivers et firmware

Certains émulateurs nécessitent que vous obteniez et utilisiez les BIOS, les pilotes ou le firmware d'origine de la console pour fonctionner correctement.

Ces fichiers contiennent du code propriétaire qui est protégé par le droit d'auteur, et les utiliser sans permission peut être illégal.

## De quel fichier ai-je besoin ?

Voici quelques détails sur les fichiers dont vous pourriez avoir besoin pour les consoles populaires :

### PlayStation 1

Pour exécuter les ISOs des jeux PlayStation 1, vous aurez besoin du fichier BIOS correspondant à la région du jeu auquel vous souhaitez jouer :

- **scph5500.bin** : BIOS Japonais
- **scph5501.bin** : BIOS Américain
- **scph5502.bin** : BIOS Européen

Voir plus de détails [ici](https://fr.wikipedia.org/wiki/Liste_des_mod%C3%A8les_de_la_PlayStation#Comparaison_des_mod%C3%A8les) et [ici](https://consolemods.org/wiki/PS1:PS1_Model_Differences).

### PlayStation 2

Pour exécuter les ISOs des jeux PlayStation 2, vous aurez besoin du fichier BIOS correspondant à la région du jeu auquel vous souhaitez jouer :

- **ps2-0250j-20100415.bin** : BIOS Japonais
- **ps2-0230a-20080220.bin** : BIOS Américain
- **ps2-0250e-20100415.bin** : BIOS Européen

Voir plus de détails [ici](https://consolemods.org/wiki/PS2:Model_Differences).

### PlayStation 3

- **PS3UPDAT.PUP (PS3 Firmware 4.92)**

Voir plus de détails [ici](https://en.wikipedia.org/wiki/PlayStation_3_models#Model_comparison).

### Xbox

- **mcpx_1.0.bin** : Image du Boot ROM MCPX
- **Complex_4627.bin** : Image du Flash ROM (BIOS)
- **xbox_hdd.qcow2** : Image du disque dur

Voir plus de détails [ici](https://xemu.app/docs/required-files/).
