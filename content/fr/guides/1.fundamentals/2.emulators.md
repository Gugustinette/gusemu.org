# Émulateurs

Il y a plusieurs types d'émulateurs, chacun conçu pour imiter une plateforme ou un système spécifique. Certains des types d'émulateurs les plus courants incluent :

- **Émulateurs de consoles** : Les émulateurs les plus populaires, qui imitent le comportement des consoles de jeux vidéo, telles que la NES/Famicom, la Game Boy, la PlayStation, et bien d'autres. Ils vous permettent de jouer à des jeux conçus pour ces consoles sur votre ordinateur.
- **Émulateurs d'arcade** : Ces émulateurs imitent le comportement des machines d'arcade, vous permettant de jouer à des jeux d'arcade sur votre ordinateur.
- **Émulateurs d'ordinateurs** : Ces émulateurs imitent le comportement des anciens systèmes informatiques, tels que le Commodore 64 ou l'Amiga, vous permettant d'exécuter des logiciels et des jeux conçus pour ces systèmes sur votre ordinateur moderne.

### À propos de RetroArch

[RetroArch](https://www.retroarch.com/) est un cas particulier, car c'est un émulateur multi-plateforme qui prend en charge une large gamme de systèmes grâce à l'API [libretro](https://www.libretro.com/), qui est une interface standardisée pour les [cœurs](/fr/guides/ressources/glossary#core) d'émulation.

RetroArch lui-même n'est pas un émulateur, mais plutôt une interface qui vous permet d'exécuter divers cœurs d'émulation, qui sont les véritables émulateurs pour différentes plateformes.

Cela est possible soit parce que les auteurs de l'émulateur ont créé un cœur libretro pour leur émulateur, soit parce que l'équipe libretro a porté l'émulateur en tant que cœur elle-même.

Vous pouvez voir la liste des cœurs pris en charge par RetroArch sur le site de libretro : https://docs.libretro.com/guides/core-list/

Les cœurs peuvent être affichés dans le tableau ci-dessus, via l'option "Afficher les cœurs Libretro".

## Émulateurs web

Avec la montée des technologies web, il existe maintenant de nombreux émulateurs qui peuvent fonctionner directement dans votre navigateur web, sans avoir besoin d'installer un logiciel.

Voici quelques exemples d'émulateurs web :

- [libretro Web](https://web.libretro.com/) : Une interface web pour exécuter les cœurs libretro directement dans votre navigateur.
- [SkyEmu Web](https://web.skyemu.app/) : Une version web de l'émulateur SkyEmu, qui prend en charge les jeux GB, GBC, GBA et DS.
- [N64 Wasm](https://www.neilb.net/n64wasm/) : Un émulateur web pour la Nintendo 64, basé sur WebAssembly.
