# Émulateurs

Il y a plusieurs types d'émulateurs, chacun conçu pour imiter une plateforme ou un système spécifique. Certains des types d'émulateurs les plus courants incluent :

- **Émulateurs de consoles** : Les émulateurs les plus populaires, qui imitent le comportement des consoles de jeux vidéo, telles que la NES/Famicom, la Game Boy, la PlayStation, et bien d'autres. Ils vous permettent de jouer à des jeux conçus pour ces consoles sur votre ordinateur.
- **Émulateurs d'arcade** : Ces émulateurs imitent le comportement des machines d'arcade, vous permettant de jouer à des jeux d'arcade sur votre ordinateur.
- **Émulateurs d'ordinateurs** : Ces émulateurs imitent le comportement des anciens systèmes informatiques, tels que le Commodore 64 ou l'Amiga, vous permettant d'exécuter des logiciels et des jeux conçus pour ces systèmes sur votre ordinateur moderne.

### À propos de RetroArch

[RetroArch](https://www.retroarch.com/) est un cas particulier, car c'est un émulateur multi-plateforme qui prend en charge une large gamme de systèmes grâce à l'API [libretro](https://www.libretro.com/), qui est une interface standardisée pour les moteurs d'émulation.

RetroArch lui-même n'est pas un émulateur, mais plutôt une interface qui vous permet d'exécuter divers moteurs d'émulation, qui sont les véritables émulateurs pour différentes plateformes.

Cela est possible soit parce que les auteurs de l'émulateur ont créé un moteur libretro pour leur émulateur, soit parce que l'équipe libretro a porté l'émulateur en tant que moteur elle-même.

Vous pouvez voir la liste des moteurs pris en charge par RetroArch sur le site de libretro : https://docs.libretro.com/guides/core-list/

La plupart des émulateurs listés dans le tableau ci-dessus ont un moteur libretro correspondant, ce qui signifie qu'ils peuvent être utilisés dans RetroArch, même sur des plateformes où ils n'ont peut-être pas de version native.
