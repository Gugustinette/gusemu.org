# ROMs

Les ROMs sont les fichiers qui contiennent les données d'un jeu vidéo. Ce sont des copies numériques des cartouches ou des disques originaux du jeu. Les ROMs sont utilisés par les émulateurs pour faire fonctionner les jeux sur du matériel moderne.

Il existe essentiellement deux types de ROMs :

- **[ROMs de cartouche](/fr/guides/ressources/glossary#rom)** : Ce sont les types de ROMs les plus courants, ils ont été dumpés de cartouches physiques utilisées pour des consoles comme la NES, la SNES, la Game Boy, et bien d'autres. Ils sont généralement au format `.nes`, `.sfc`, `.gb`, etc., selon la console pour laquelle ils ont été publiés.
- **[ISOs](/fr/guides/ressources/glossary#iso)** : Ce sont des ROMs qui ont été extraits de disques optiques, tels que les CD ou les DVD. Ils sont typiquement utilisés pour des consoles comme la PlayStation, la Sega CD, la GameCube et d'autres. Ils sont généralement au format `.iso`, `.bin`, ou `.cue`.

Bien que le terme "ROM" soit souvent utilisé pour désigner les deux types, techniquement, seuls les dumps de cartouche sont de véritables ROMs, tandis que les ISOs seraient plus précisément décrits comme des images de disque.

Cependant, dans le contexte de l'émulation, les deux types de fichiers sont couramment appelés ROMs. Dans cette documentation, nous utiliserons le terme "ROM" pour désigner à la fois les dumps de cartouche et les images de disque, sauf indication contraire.

## Formats compressés

Dans certains cas, les ROMs peuvent être distribués dans des formats compressés personnalisés pour économiser de l'espace.

Voici quelques formats compressés populaires que vous pourriez rencontrer :

- `.chd` : PlayStation, Sega CD/Saturn/Dreamcast, et d'autres.
- `.rvz` : Gamecube et Wii.
- `.wua` : Wii U

La plupart du temps, vous ne devriez pas avoir à vous soucier de ces formats, car la plupart des émulateurs peuvent les lire directement. Cependant, si vous avez besoin de les convertir en un format plus courant (par exemple, pour appliquer des patchs), voici quelques outils que vous pouvez utiliser :

- Pour les fichiers `.chd`
  - `chdman` via [`rom-tools`](https://formulae.brew.sh/formula/rom-tools)
  - [`bchunk`](https://formulae.brew.sh/formula/bchunk)
- Pour les fichiers `.rvz`, utilisez l'outil de conversion intégré de [Dolphin](https://dolphin-emu.org/)
- Pour les fichiers `.wua`, utilisez l'outil de conversion intégré de [Cemu](https://cemu.info/)
