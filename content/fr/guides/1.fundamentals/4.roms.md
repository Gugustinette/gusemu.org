# ROMs

Les ROMs sont les fichiers qui contiennent les données d'un jeu vidéo. Ce sont des copies numériques des cartouches ou des disques originaux du jeu. Les ROMs sont utilisés par les émulateurs pour faire fonctionner les jeux sur du matériel moderne.

Il existe essentiellement deux types de ROMs :

- **[ROMs de cartouche](/fr/guides/ressources/glossary#rom)** : Ce sont les types de ROMs les plus courants, ils ont été dumpés de cartouches physiques utilisées pour des consoles comme la NES, la SNES, la Game Boy, et bien d'autres. Ils sont généralement au format `.nes`, `.sfc`, `.gb`, etc., selon la console pour laquelle ils ont été publiés.
- **[ISOs](/fr/guides/ressources/glossary#iso)** : Ce sont des ROMs qui ont été extraits de disques optiques, tels que les CD ou les DVD. Ils sont typiquement utilisés pour des consoles comme la PlayStation, la Sega CD, la GameCube et d'autres. Ils sont généralement au format `.iso`, `.bin`, ou `.cue`.

Bien que le terme "ROM" soit souvent utilisé pour désigner les deux types, techniquement, seuls les dumps de cartouche sont de véritables ROMs, tandis que les ISOs seraient plus précisément décrits comme des images de disque.

Cependant, dans le contexte de l'émulation, les deux types de fichiers sont couramment appelés ROMs. Dans cette documentation, nous utiliserons le terme "ROM" pour désigner à la fois les dumps de cartouche et les images de disque, sauf indication contraire.

## Formats compressés

Dans certains cas, les ROMs peuvent être distribués dans des formats compressés personnalisés pour économiser de l'espace.

Voici quelques formats compressés populaires que vous pourriez rencontrer :

- `.chd` : PlayStation, Sega CD/Saturn/Dreamcast, et d'autres.
- `.rvz` : Gamecube et Wii.
- `.wua` : Wii U

La plupart du temps, vous ne devriez pas avoir à vous soucier de ces formats, car la plupart des émulateurs peuvent les lire directement. Cependant, si vous avez besoin de les convertir en un format plus courant (par exemple, pour appliquer des patchs), voici quelques outils que vous pouvez utiliser :

- Pour les fichiers `.chd`
  - `chdman` via [`rom-tools`](https://formulae.brew.sh/formula/rom-tools)
  - [`bchunk`](https://formulae.brew.sh/formula/bchunk)
- Pour les fichiers `.rvz`, utilisez l'outil de conversion intégré de [Dolphin](https://dolphin-emu.org/)
- Pour les fichiers `.wua`, utilisez l'outil de conversion intégré de [Cemu](https://cemu.info/)

## Où trouver des ROMs

Il y a deux façons principales d'obtenir des ROMs : le téléchargement direct et le torrenting. Les deux ont leurs avantages et leurs inconvénients, mais ne vous inquiétez pas, nous allons parler des deux ici.

Cependant, vous devez d'abord comprendre certains enjeux de sécurité et de légalité avant de télécharger des ROMs depuis Internet, alors nous allons couvrir cela en premier.

### Sécurité et légalité

Comme le téléchargement de ROMs depuis Internet peut être une zone grise légale (et que vous ne pouvez pas faire confiance à votre [FAI](https://fr.wikipedia.org/wiki/Fournisseur_d%27acc%C3%A8s_%C3%A0_Internet)), il est recommandé d'utiliser un VPN lorsque vous le faites.

Évitez les VPN gratuits, car ils ont souvent une sécurité médiocre et peuvent même vendre vos données. Au lieu de cela, envisagez d'utiliser un service VPN payant réputé qui a une politique de confidentialité solide et ne conserve pas de [logs](/fr/guides/ressources/glossary#logs)).

### Téléchargement direct

Le téléchargement direct est la façon la plus simple d'obtenir des ROMs. Vous trouvez simplement un site Web qui héberge les ROMs que vous voulez, et vous les téléchargez directement à partir de là.

Le site Web n'a généralement aucun moyen de savoir qui vous êtes, mais votre FAI peut voir que vous téléchargez des fichiers à partir de ce site Web, c'est pourquoi l'utilisation d'un VPN est recommandée.

Voici quelques sources populaires pour les ROMs :

- [Vimm's Lair](https://vimm.net/vault)
- [LoLROMs](https://lolroms.com/)
- [EmuParadise](https://www.emuparadise.me/)
- [Retrostic](https://www.retrostic.com/)
- [rom hustler](https://romhustler.org/)
- [TheOldComputer](https://www.theoldcomputer.com/roms/)
- [RomUlation](https://www.romulation.org/)
- [ROMsNS](https://romsns.com/)
- [SwitchRom.net](https://switchrom.net/)
- [NXBrew.net](https://nxbrew.net/)
- [r/ROMs Megathread](https://r-roms.github.io/)
- Archive.org (vitesse de téléchargement lente)
  - [3ds-decrypted-roms312com](https://archive.org/download/3ds-decrypted-roms321com/all/)
  - [ns-nsp-collection](https://archive.org/download/ns-nsp-collection)

Gardez à l'esprit que nous ne garantissons pas l'intégrité ou la légalité des ROMs trouvés sur ces sites, alors assurez-vous de faire vos propres recherches avant de télécharger quoi que ce soit.

### Torrenting

[BitTorrent](https://fr.wikipedia.org/wiki/BitTorrent) est un protocole de partage de fichiers peer-to-peer qui permet aux utilisateurs de partager des fichiers entre eux.

Lorsque vous torrentez une ROM, vous la téléchargez à partir d'autres utilisateurs qui ont la même ROM sur leur ordinateur. C'est une distinction très importante à faire, car cela signifie que lorsque vous torrentez une ROM, vous vous exposez directement à d'autres utilisateurs.

Les sites basés sur le torrenting vous donnes généralement soit :

- Une fichier torrent (avec l'extension `.torrent`), qui est un petit fichier qui contient des informations sur la ROM que vous voulez télécharger, et les trackers qui vous aideront à trouver d'autres utilisateurs qui ont la même ROM.
- Un lien magnet, qui est une URL qui contient les mêmes informations qu'un fichier torrent, mais sous une forme plus compacte.

Pour utiliser les fichiers torrent ou les liens magnet, vous avez besoin d'un client torrent. Voici quelques-uns des plus populaires :

- [qBittorrent](https://www.qbittorrent.org/)
- [uTorrent](https://www.utorrent.com/)
- [Transmission](https://transmissionbt.com/)
- [BitTorrent](https://www.bittorrent.com/)
